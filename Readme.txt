Оипсание приложения: данное приложение предназначено для жителей (для классификации жалоб)
Возможности приложения: классификация жалоб, получение справочной информации, возможность связи с Разработчиком
Как работает: запуск GUI -> считываение текста -> отправка запроса на сервер(API) -> обработка запроса(на сервере) -> получение Предсказания
Инструкция: 1. запустить  server.py, 2. запустить tk_app.py, 3. ввести текс, 4. получить Предсказание
